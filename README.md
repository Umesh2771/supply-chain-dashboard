## 🚀 Project Overview
This project demonstrates an end-to-end data pipeline using:
- MySQL (Database)
- Python (Data extraction)
- Power BI (Visualization)

## 🔧 Tech Stack
- MySQL
- Python (pandas, sqlalchemy)
- Power BI

## 📂 Dataset
Product dataset including:
- product_id
- product_name
- category
- sku
- unit_cost
- selling_price

## 📊 Dashboard Features
- Products by Category (Bar Chart)
- Category Distribution (Pie Chart)
- Total Products KPI

## 🔄 Workflow
1. Data stored in MySQL
2. Extracted using Python
3. Loaded into Power BI
4. Visualized using charts
